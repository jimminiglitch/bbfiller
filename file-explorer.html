File Explorer Integration Instructions

Option 1: Add to your existing site

  1. Copy the file-explorer.js file to your /js folder
  2. Copy the file-explorer.css file to your /css folder
  3. Add the following HTML to your site where you want the file explorer window to appear: 
    <div id="file-explorer" class="popup-window">
  <div class="window-header">
    <span>FILE_EXPLORER.EXE</span>
    <button class="minimize">_</button>
    <button class="maximize">▭</button>
    <button class="close">X</button>
  </div>
  <div class="window-content file-explorer-content">
    <!-- File explorer content will be loaded here by file-explorer.js -->
  </div>
</div>
  4. Add this script tag at the end of your body: 
    <script src="js/file-explorer.js"></script>

Option 2: Test standalone version

You can also test the file explorer by opening this HTML file directly in your browser.
